## 代码高亮与格式测试

我们可以嵌套测试内联代码：`console.log('Hello World')`，以及测试多行代码块：

```javascript
// 测试 JavaScript 代码块
function sayHello(name) {
  return `Hello, ${name}! Welcome to Astro and Tailwind v4.`;
}

console.log(sayHello("PaperAir"));
```
